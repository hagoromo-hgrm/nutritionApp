# Imported食品データ

このディレクトリには、ユーザー提供または事前検証済みの食品DBを、アプリの`Food`形式へ正規化した生成物だけを置きます。元ファイルはGitへ含めません。

## 購入予測商品DB

`purchase_prediction_foods.json`は、ユーザー提供の`nutrition-pwa-food-db`形式JSONを次のコマンドで変換したものです。

```bash
python3 scripts/import_food_database.py <input-json> data/imported/purchase_prediction_foods.json
```

- 入力件数: 255件
- 採用件数: 144件
- 除外件数: 111件
- 除外対象: 基準量・基準単位がなく、全栄養値が`null`の名称候補
- 由来: メーカー公式、流通事業者、商品ラベル転記DB、文部科学省食品成分データベース等を含むユーザー提供DB
- 取得日: 2026-07-23

変換時には、IDを`imported:`名前空間へ移し、`source`を`imported`へ変更します。空のバーコード・メーカー等をアプリの型へ合わせ、基準単位と重複する入力単位換算は削除します。入力カテゴリは食品のバリエーション概念ではないため、各商品を独立した食品グループへ割り当てます。

元DBのメタデータにあるライセンス文言は生成物へ保持します。参照元ごとの利用条件を一括して確認できる情報は元DBに含まれていないため、この生成物は個人利用を前提とし、再配布前には各参照元の条件を別途確認してください。
